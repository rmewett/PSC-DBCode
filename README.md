**PSC-DBCode (2005)** by Richard Mewett <br>
A submission for Planet Source Code <br>

This utility is used to script a database. <br>

You can generate SQL Script for Query Analyser which is useful for migrating a JET (Access) database to SQL Server. <br>

You can also generate VB Source code to create a target database via DAO objects or DAO SQL Execute statements. <br>

I wrote this VERY quickly a few years back for a SQL migration project. I will update for ADO if anyone  interested! <br>

⭐ Features <br>
* Generate SQL Script for Query Analyser
* Generate VB Source code to create a target database via DAO objects or DAO SQL Execute statements
